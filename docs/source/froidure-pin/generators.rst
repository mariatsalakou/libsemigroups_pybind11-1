Generators
----------

This page contains information about the methods of the :py:class:`FroidurePin`
class that relate to generators.

.. py:method:: FroidurePin.add_generator(self: FroidurePin, x: Element) -> None

   Add a copy of an element to the generators.

   :param x: the generator to add.
   :type x: Element

   :return: None
                 
.. py:method:: FroidurePin.add_generators(self: FroidurePin, coll: List[Element]) -> None

   Add collection of generators.

   :param coll: the collection of generators to add.
   :type coll: List[Element]

   :Returns: None

.. py:method:: FroidurePin.closure(self: FroidurePin, coll: List[Element]) -> None

   Add non-redundant generators in collection.

   :param coll: the collection of generators to add.
   :type coll: List[Element]

   :Returns: None
                
.. py:method:: FroidurePin.copy_add_generators(self: FroidurePin, coll: List[Element]) -> FroidurePin 

   Copy and add a collection of generators.

   :param coll: the collection of generators to add.
   :type coll: List[Element]

   :Returns:
      A new ``FroidurePin`` instance generated by the
      generators of ``self`` and ``coll``.

.. py:method:: FroidurePin.copy_closure(self: FroidurePin, coll: List[Element]) -> FroidurePin 

   Copy and add non-redundant generators.

   :param coll: the collection of generators to add.
   :type coll: List[Element]

   :return:
     A new ``FroidurePin`` instance generated by the generators of ``self`` and
     the non-redundant generators in ``coll``.

.. py:method:: FroidurePin.number_of_generators(self: FroidurePin) -> int

   Returns the number of generators.

   :Parameters: None
   :return: An ``int``.

.. py:method:: FroidurePin.generator(self: FroidurePin, i: int) -> Element

   Returns the generator with specified index.

   :param i: the index of a generator.
   :type i: int

   :return: An element.
